What's New?

1.90a

* 5 breaking changes introduced:

LoadSound - Load3DSound has been removed, use LoadSound with the 3D flag instead
GetListener - CreateListener has been removed, use GetListener instead
PlayMusic - Removed, use LoadSound, LoopSound and PlaySound instead
SoundPan - Removed
ChannelPan - Removed

* 17 new commands introduced:

Ptr - new command to cast types to pointers
Strict - new command to enforce strict type checking
StreamSound - new command to stream sound from files instead of loading them entirely in memory
ErrorLog - new command to get the last internal error as a string
GetLineTrace - new command to get the full stack trace as a string
TextureBumpEnvMat - direct d3d abstraction of D3DTSS_BUMPENVMATXX
TextureBumpEnvScale - direct d3d abstraction of D3DTSS_BUMPENVLSCALE
TextureBumpEnvOffset - direct d3d abstraction of D3DTSS_BUMPENVLOFFSET
TextureLodBias - direct d3d abstraction of D3DSAMP_MIPMAPLODBIAS
SoundRange - set the near and far bounds a sound is played
ChannelRange - set the near and far bounds a specific channel is played
ChannelPos - set the 3d position and velocity of a specific channel
ChannelSeek - new command to seek a channel to a specific position
NoTrace - new command to disable stack tracing
CopyDir - new command to copy an entire directory recursively
Test - new command to dictate a Test function
Assert - new command to assert a statement is true

* 4 new capabilities:
Memory Access Violations - Replaced with descriptive errors. If in debug mode true Exceptions are thrown, if running in release mode errors are handled gracefully and accessible via ErrorLog without disrupting program execution.
3D Sound - 3D sounds can be given a position and velocity directly and don't have to be attached to an entity. See PlaySound and ChannelPos
Sound Looping - can be toggled on and off. See LoopSound
ExecFile - improved to specify params and a working directory as well has return process handle.
