FLOWER - 
Cluster Amount       - This is the amount of flower patches you desire to be included in this model set ( 1= one flower patch ) ( max 10 )
Stems Per Cluster   - This determines how many stems each flower will contain per cluster. (ex: 3 would have 3 stems per cluster)
Cluster Seperation   - The distance between each cluster in the model
Stem Seperation     -  The distance between each stem in the cluster.
